## ðŸ”¹ Project Description
**Car Price Prediction System** is a **Streamlit web app** that predicts the **future price of a car** using **Machine Learning (Linear Regression)**.  

It helps users to:
- Predict the current price of a car
- Visualize depreciation trend over years
- Find the best time to sell
- Get AI-generated price analysis
- Download PDF report with car details, graph, and advice

The system supports both **desktop** and **mobile view**, with **mobile view providing step-by-step automatic flow** for easy usage.

---

## ðŸ”¹ Features

| Feature | Description |
|---------|-------------|
| Price Prediction | Predict current car price in lakhs |
| Depreciation Graph | Shows price trend over next 5 years |
| Best Time to Sell | Suggests the optimal selling year |
| AI Explanation | Generates short analysis using Gemini AI |
| Download PDF | Generate PDF report with all details |
